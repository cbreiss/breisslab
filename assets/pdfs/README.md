# PDFs Directory

Store PDF files here (publications, CVs, etc.)

## How to Add PDFs

1. Upload PDF files to this directory
2. Link to them in your pages:
   ```markdown
   [PDF](/breisslab/assets/pdfs/filename.pdf)
   ```

## Recommended Files

- Publications (e.g., `smith2024.pdf`)
- CV/Resume (e.g., `pi-cv.pdf`)
- Posters (e.g., `conference2024-poster.pdf`)
- Protocols or methods papers

**Note:** Only upload PDFs you have rights to distribute!
